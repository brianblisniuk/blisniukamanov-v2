# 📋 Plantilla para sumar un viaje nuevo

Esta es la plantilla que llenás vos con los datos reales del viaje. Cuando esté completa, me la pasás y yo la convierto en una página dentro del sitio en menos de 30 minutos (queda con URL propia, mapa interactivo, fechas, lodges, etc).

**Cómo usar esta plantilla:**

1. Copiá este archivo a un Google Doc o un editor de texto.
2. Reemplazá cada `__________` con la información real. Si un campo no aplica, dejalo en blanco y avisame.
3. Cuando termines, mandámelo (por mail / WhatsApp / lo pegás en el chat).
4. Yo lo convierto al formato del generador y se publica.

Si dudás de algún campo, **mirá el viaje "La Gran Migración" (`gran-migracion.html`) en el sitio actual como referencia** — es el modelo del que se sacan los demás.

## 🚀 Cómo se carga al sitio

Una vez completada la plantilla:

1. Me la pasás (mail, WhatsApp, chat).
2. Yo agrego los datos al `tools/build.py` (un objeto Python con todos los campos).
3. Corro `python3 tools/build.py` y se genera la página HTML automáticamente.
4. Git commit + push → Netlify redeploya en 30 segundos.
5. La página queda en `blisniukamanov.com/SLUG`.

**Tiempo total de mi lado:** 20-30 min por viaje.

Cuando esté la integración del CMS más avanzada, vas a poder hacer esto directamente desde `/admin/` sin pasar por mí.
